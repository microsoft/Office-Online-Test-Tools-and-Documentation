
Glossary
========

..  glossary::
    :sorted:

    WOPI client
        A WOPI client is an application or service that executes WOPI operations on a WOPI server by issuing HTTP
        requests to the WOPI server's WOPI endpoints. Office Online is an example of a WOPI client.

    WOPI host
    WOPI server
        A WOPI server, often called a WOPI host or just simply a *host*, is an application or service that implements
        WOPI endpoints and operations as described in this documentation, such as :ref:`CheckFileInfo` and
        :ref:`GetFile`. OneDrive for Business is an example of a WOPI host.
